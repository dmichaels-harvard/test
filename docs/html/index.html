Overview

smaht-submitr: A remote file submitter for SMaHT

Description

The smaht-submitr is a command-line tool for uploading files to SMaHT. It is implemented as Python package and distributed on PyPi here: smaht-submitr

System Requirements

  • python (>= 3.8 and <= 3.11)

  • pip (>= 22.0.0)

  • poetry (>= 1.4.0)

  • smaht-portal (>= 1.0.0)

What To Do Next

Advanced users who have already installed Python can proceed to instructions for Installing smaht-submitr.

Less experienced users should start with instructions for Installing Prerequisites, which will introduce some basics for working with the terminal and installing the dependencies to run this package.
